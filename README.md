## 專案套件初始化

```bash
npm i
```

### 本地開發環境

```bash
npm run dev
```

### 打包專案

```bash
npm run build
```

### 預覽打包的專案

```bash
npm run preview
```

### node 版本

```bash
v20.13.1
```

### 多國語系

1. 翻譯文件：https://docs.google.com/spreadsheets/d/17zSN3HM10OMrVGKpI1__o7IL0fej8kt6wm3289WmpJI/edit?usp=sharing
2. 將翻譯文件下載為.xlsx檔案，並改名成xlsx.xlsx
3. 將xlsx.xlsx丟到根目錄
4. 執行

```bash
npm run xlsx
```
